jenkins 小技巧

设置项目构建的描述信息,pipline 的情况下这样设置

stage('xxx'){
    steps{
        script {
            currentBuild.name ="MY_VERSION_NUMBER"
            currentBuild.description ="MY_PROJECT MY_VERSION_NUMBER"
        }
    }
}

在自由风格的情况下，需要安装 ‘description setter plugin’, 然后在项目的时候下面添加一个编译后的步骤，选择 ‘Set build description’, 然后 填写上正则表达式，捕获日志中的字符串，在 ‘Description’ 中可以用 \number 的方式引用正则的捕获。
