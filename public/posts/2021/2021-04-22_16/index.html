coredns 创建泛域名解析

例子如下:

.:53 {
    errors
    health {
      lameduck 5s
    }
    ready
    template IN A test.cn {
      match .*\.test\.cn
      answer "{{ .Name }} 60 IN A 192.168.1.1"
      fallthrough
    }
    forward . /etc/resolv.conf
    cache 30
    reload
    loadbalance
}

这里创建了一个 "*.test.cn" 的泛域名解析，所有的解析都被指向了地址 "192.168.1.1"
