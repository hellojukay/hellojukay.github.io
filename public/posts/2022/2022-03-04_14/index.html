tmux 相关命令

默认情况下， tmux 的前缀是 ctrl + b， 下面是一些常用的命令：

  1. tmux new -s 启动 tmux 并且创建一个会话
  2. tmux ls 列出 tmux 的所有会话
  3. tmux send-keys -t 向 tmux 会发发从按键
  4. prefix + d detach 当前会话
  5. prefix + attach -t attach 到一个会话中
  6. prefix + " 上下分割窗口右
  7. prefix + % 左右分割窗口
  8. prefix + c 再当前会话创建新的窗口
  9. prefix + x 关闭当窗口
  10. prefix + s 列出所有的会话
  11. prefix + w 列出所有的窗口
  12. prefix + , 重新命名当前窗口
  13. prefix + n 切换到下一个窗口
  14. prefix + p 切换到上一个窗口
